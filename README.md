# NestSteps Parent Companion Prototype

A mobile-first Next.js prototype for parents guiding short fine motor activities at home.

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Prototype Coverage

- Onboarding: parent signup, child profile, simple developmental questions, daily routine
- Dashboard: today’s session, streak, progress, upcoming activities
- Activity: preparation, guided steps, timer, pause/resume, help, child refusal options
- Completion: session summary, encouragement, progress explanation
- Return next day: gentle next-day re-entry with a new low-prep activity

The UI uses Tailwind and local shadcn/ui-style primitives in `components/ui`.
